# Personal Portfolio Website

Hi, I’m **Yadhir Ramrethan**, and this is my personal portfolio website built using **HTML, CSS, and JavaScript**.  
I created this project as part of my **Future Interns Full Stack Web Development internship task** to build a professional online portfolio that showcases my skills, projects, background, and contact details.

This website acts as my **digital resume and proof of work**, where recruiters, clients, and hiring managers can learn more about me and the projects I have worked on.

## Live Demo
[View Portfolio](https://yadhirr.github.io/FUTURE_FS_01/)

## Project Purpose
The purpose of this project is to create a clean, responsive, and professional portfolio website that reflects my journey as a student developer. It highlights my front-end development skills, practical project work, and career goals in a way that is easy to view on both desktop and mobile devices.

## Features
- Responsive design for mobile, tablet, and desktop
- Professional homepage introduction
- About section with background and goals
- Skills and tech stack section
- Projects section showcasing my work
- Resume snapshot section
- Contact section with links and form validation
- Interactive navigation and project filtering
- SEO-friendly HTML structure
